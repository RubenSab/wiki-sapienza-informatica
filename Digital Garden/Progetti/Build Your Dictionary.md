# Overview

- Python tool that lets you build and edit flexibly your own dictionary.

# What can it do?

- Get words definition, translation in other languages and pronunciation from Wikipedia free dictionary (or insert your own) and edit them.
- Make an alphabetically ordered spreadsheet with all the words, definitions, pronunciations and different languages.
- Highlight noun-adjective-verb-adverb clusters.
- Write annotations.
- Make flashcards and get automatically generated quizzes.
- Discover new words + word of the day.
- Track how many notes you insert over time, generate graphs.

# Goal

- Include only the tool features i'm interested with, to make it either a lightweight notebook or a fully-fledged computer assisted language learning tool.

# Skills required

- file handling
- json handling
- markdown formatting
- language translation
- web scraping
- ui

1. Get all user inputs (word list)
