> In [[RISC-V]], l'istruzione Immediato o istruzione I-type, è un'istruzione che opera con 3 [[registri]]. (2 argomenti e 1 risultato)

- **immediate** (12 bit): function code (additional op-code). NON è un registro, ma una parte dell'**istruzione**, però rappresenta un numero/informazione contenuto nell'istruzione stessa, non in un registro
- **rs1** (5 bit): first source [[registri|register]] number
- **func3** (3 bit): function code(additional op-code)
- **rd** (5 bit): register destination number
- **opcode** (7 bit): operation code
