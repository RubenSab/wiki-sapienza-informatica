> In [[RISC-V]], l'istruzione registro o istruzione R-type, è un'istruzione che compie un'operazione aritmetico/logica, NON ha accesso alla [[memoria]]; opera con 3 [[registri]]. (2 argomenti e 1 risultato)

- **funct7** (7 bit): function code (additional op-code)
- ***rs2*** (5 bit): second source register number
- ***rs1*** (5 bit): first source register number
- **funct3** (3 bit): function code (additional op-code)
- ***rd*** (5 bit): register destination number
- **opcode** (7 bit): operation code
