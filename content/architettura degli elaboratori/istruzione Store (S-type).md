> In [[RISC-V]], l'istruzione Store o istruzione S-type, è un'istruzione che scrive in [[memoria]]; opera con 3 registri. (2 argomenti e 1 risultato)
 
- **Imm\[11:5\]** (7 bit): function code (additional op-code)
- ***rs2*** (5 bit): second source [[registri|register]] number
- ***rs1*** (5 bit): first source register number
- **funct3** (3 bit): function code (additional op-code)
- **Imm\[4:0\]** (5 bit): register destination number
- **opcode** (7 bit): operation code
