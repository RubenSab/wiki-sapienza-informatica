# Addizione

$f: N \times N \to N$
$(n,m) \to n + m$

sposta di n passi da m ([[algoritmo]])

- proprietà commutativa
- 0 elemento nullo

# Moltiplicazione

$f: N \times N -> N$

$(n,m) \to n \times m = \sum_{i=1}^{m}n$ (funzione)

```python
p = 0
for i in range(m): 
	p+=n
print(p)
```

(algoritmo)

- proprietà commutativa
- 1 elemento nullo

Non esiste un numero intero tale che $n = a-b : b>a$, per la sottrazione bisogna introdurre gli [[numeri interi relativi]].
