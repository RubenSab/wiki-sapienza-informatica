# 1.

> L'insieme $\{x \in \mathbb{Z}: x \equiv 3 \mod 7 \land x \equiv 1 \mod 5\} \subset \mathbb{Z}$

1. è vuoto: falso, perché contiene 4 (conseguenza del teorema cinese dei resti, 7 e 5 sono primi fra loro)
2. contiene 1 (falso, perché $1 \not\equiv 3 \mod 7$)
3. è un sistema di rappresentanti $\mod 35$ (falso, perché il SCR contiene 1)
4. è non vuoto (vero, perché contiene 4)

# 2.

> Sia $G$ un gruppo in notazione moltiplicativa $H<G$.

1. Se $H$ è contenuto in tutti i sottogruppi di $G$ allora $H = \emptyset$ (falso per definizione di gruppo)
2. Se per ogni $x \in G$ e $h \in H$ si ha $xh = hx$ allora $G$ è abeliano (falso)
3. Se $H \neq {1_{G}}$ allora $H=G$ (falso)
4. $G$ abeliano allora $\forall x \in G$ e $h \in H\ \exists h': xh = hx$ (vero)

# 3.

> $\forall \sigma \in S_{n}$

1. $\exists n \geq 1: \sigma^{n} = 1_{S_{n}}$ (vera)
2. $\sigma^{n}=1_{S_{n}}$ (falso)
3. $\sigma$ è un ciclo (falso)
4. $\sigma$ si decompone in prodotto di $n$ trasposizioni (falso)

# 4.

>  $n \in \mathbb{Z}$ arbitrario. $n, n+2$ sono:

1. sempre primi fra loro (falso)
2. tale che il loro MCD sia pari (falso)
3. sempre simultaneamente divisibili per almeno un primo (vero)
4. tale che il loro MCD $\mid 2$ (vero)

# 5.

> $G$ è un gruppo, siano $H, H' < G$

1. $H \cap H' < G$ (vero)
2. $H \cup H' < G$ (falso)
3. se $H \cap H' < G$ allora $H \cap H = \emptyset$ (falso)
4. se $H \cap H' < G$ allora $H \cap H'$ singleton (falso)

# 6.

> $H < G, G'$

1. $H \approx G'$
2. $\frac{G'}{I}$ è ben definito ed è isomorfo al quoziente
3. $I < G$ isomorfo ad $H$
4. $\forall g \in G\ (g^{-1} Hg=H)$ (vero)
