# Unione

>$A \cup B = \{x\ |\ x \in \text A \text{ oppure } x\in B\}$
# Intersezione

>$A \cap B = \{x\ |\ x \in A \text{ e } x \in B\}$

## Insiemi disgiunti

^0004d7

> Due insiemi $A$ e $B$ sono disgiunti se ($A \cap B = \emptyset$)

> N.B.: $\emptyset$ è disgiunto da qualsiasi insieme ($A \cap \emptyset = \emptyset$)

> N.B.: Tutti gli insiemi contengono almeno 2 insiemi disgiunti, anche l'insieme vuoto

# Differenza

>$A-B = \{x\ |\ x \in A,\ x \notin B\}$
# Complemento

>$\overline{A} = \{x\in U\ |\ x\notin A\}$

Considerando un sottoinsieme di A, il complemento è l'insieme di tutti gli elementi rimanenti di A. Il [[complementazione]] è l'operatore $\overline A$  dell'interpretazione 'ad aree' dell'algebra di Boole.   
# Prodotto cartesiano

> Il prodotto cartesiano ($\times$) di due insiemi $A$ e $B$ è l'insieme di tutte le coppie possibili di A e B, le quali **NON** sono insiemi ma coppie ordinate.

>$A \times B = \{(a,b)\ ∣\ a\in A,\ b \in B\}$

> $|A| = |a_{1} \times a_{2} \times \ldots \times a_{n}| = \prod_{k=1}^{n}{|a_{k}|}$

Si noti che ogni punto del piano cartesiano è un elemento dell'insieme $A\times B$.
